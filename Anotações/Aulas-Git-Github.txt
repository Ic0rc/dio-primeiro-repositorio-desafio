# Comandos para GIT BASH

- CD (ENTRA NO DIR.) || CD .. (SAI DO DIR.)
- LS ou LS -A (MOSTRA OCULTOS)
- GIT INIT (CRIA-SE O REPOSITÓRIO)
- GIT ADD * (TODOS)
- GIT COMMIT -M "AQUI VAI A MENSAGEM"
- GIT STATUS
- MV ARQUIVO.TXT ./PASTA/ (MOVE ARQUIVO PARA PASTA)
- ECHO > README.MD (CRIA ARQUIVO README)
- GIT REMOTE ADD ORIGIN HTTPS:// (LINKA REPOSITÓRIO GITHUB AO GIT)
- GIT CLONE HTTP:// (LINK DO REPOSITÓRIO NO GITHUB)
